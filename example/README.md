# Stage 3D Example

This example builds a small interactive scene with:

- a Filament viewport;
- a GLB fox model;
- directional and movable point lights;
- a static physics floor;
- a dynamic compound collider synchronized with the model;
- automatic playback of the first GLB animation clip when available;
- two camera presets that can be switched with `FilamentViewportController`.

Run it from this directory:

```sh
flutter run
```
